# technotes: 技術的なブログ

## 使い方

1. drafts/ に準備稿を markdown で記事を書いてください。

```
drafts/
      YYYY-MM-DD-TITLE.md
      .... 

1. リリースしてよくなったら、docs/_post/ に git mv してください

```
% git mv drafts /YYYY-MM-DD-TITLE.md docs/_post/
% git commit
% git push
```
